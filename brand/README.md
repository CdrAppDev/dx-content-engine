# Brand context

Source of truth for theme, positioning, and voice. Every content skill loads
these files at runtime, so editing them changes every piece of generated
content immediately — no rebuild needed.

Three files, three jobs:

- **theme.md** — the overarching narrative. What you're saying across everything.
- **positioning.md** — who it's for, what makes it different, what to claim.
- **voice.md** — how sentences sound. Tone, rhythm, words to use and avoid.

Fill these in before generating content. Generated work without filled-in
brand context will be generic.
